tryUtf8

tryUtf8: boolean = false

When set to true, the decoder will attempt to decode RESP Bulk strings (which are binary strings, i.e. Uint8Array) as UTF-8 strings. If the string is not valid UTF-8, it will be returned as a Uint8Array.

You can toggle this setting at any time, before each call to decode() or read(), or other methods.
